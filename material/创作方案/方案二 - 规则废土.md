# 方案二：《规则废土》

> **题材**：废土末日 + 都市异能 + 诡异怪谈
> **定位**：末日生存/规则怪谈/异能觉醒/基地经营
> **目标字数**：300-500 万字

---

## 一、核心设定

### 1.1 一句话故事
> 2030 年，"规则怪谈"降临现实，全球 90% 人类消失。幸存者发现，只有遵守或利用规则，才能在这个诡异废土上活下去。

### 1.2 核心冲突
- **表层**：幸存者 vs 规则怪谈区域
- **里层**：人类 vs 诡异源头
- **深层**：秩序 vs 混乱（重建文明）

### 1.3 世界观

**灾难设定——"规则入侵"**：
```
2030 年 7 月 15 日，"那一天"降临：
- 全球突然出现无数个"规则区域"
- 每个区域都有独特的"杀人规则"
- 违反规则者，会诡异死亡
- 90% 人类在第一个月消失

剩余人类分布：
- 安全区（无规则区域）：5%
- 流浪者（移动躲避）：3%
- 沦陷区（已死亡/异化）：2%
```

**规则怪谈类型**：

| 类型 | 特点 | 危险等级 | 示例 |
|------|------|---------|------|
| **空间类** | 特定区域有规则 | ★★★ | "电梯不能按 4 楼" |
| **时间类** | 特定时间有规则 | ★★★★ | "午夜不能照镜子" |
| **行为类** | 特定行为触发 | ★★★ | "不能回头" |
| **语言类** | 特定话语触发 | ★★★★★ | "不能说出它的名字" |
| **认知类** | 知道即触发 | ★★★★★★ | "不要看月亮" |
| **连锁类** | 多重规则组合 | ★★★★★★ | "七重死亡规则" |

**危险等级划分**：
```
★ 安全：无规则或规则温和
★★ 普通：规则明确，易规避
★★★ 危险：规则复杂，有陷阱
★★★★ 高危：规则有迷惑性，死亡率高
★★★★★ 灾难：规则几乎无法完全遵守
★★★★★★ 绝望：无解规则，只能逃离
★★★★★★★ 诡异源头：规则发源地
```

## 二、异能体系

### 2.1 异能觉醒

**觉醒条件**：
1. 在规则区域存活超过 7 天
2. 经历至少一次"规则杀"但幸存
3. 有一定精神韧性（未崩溃）

**异能本质**：
- 人类大脑适应规则入侵后进化的能力
- 可以"感知"规则、"利用"规则、甚至"修改"规则
- 但每次使用都会积累"诡异侵蚀度"

### 2.2 异能分类

| 异能类型 | 核心能力 | 代表能力 | 战斗风格 |
|---------|---------|---------|---------|
| **感知系** | 感知规则、危险预警 | 规则视觉、危险感知 | 辅助/侦查 |
| **规则系** | 利用规则、规则免疫 | 规则漏洞、短暂免疫 | 智斗/破解 |
| **精神系** | 精神攻击、意识操控 | 精神冲击、意识入侵 | 控制/秒杀 |
| **强化系** | 身体强化、超速再生 | 力量强化、断肢再生 | 近战/坦克 |
| **元素系** | 操控自然元素 | 火焰、雷电、冰霜 | 远程/群攻 |
| **召唤系** | 召唤诡异/物品 | 召唤帮手、道具具现 | 多变的 |
| **空间系** | 空间操控、瞬移 | 空间切割、短距瞬移 | 机动/刺杀 |
| **因果系** | 因果操控、概率修改 | 必中、厄运诅咒 | 规则级 |

### 2.3 异能等级

```
等级划分（由低到高）：

F 级：基础异能（微弱效果）
  └── 例：微弱危险感知

E 级：普通异能（实用效果）
  └── 例：清晰危险感知、小型火焰

D 级：精良异能（战斗可用）
  └── 例：火焰操控、力量×3 强化

C 级：精英异能（战术价值）
  └── 例：大规模火焰、短距瞬移

B 级：大师异能（战略价值）
  └── 例：火焰领域、空间切割

A 级：宗师异能（城市级）
  └── 例：天火降临、空间囚笼

S 级：传奇异能（文明级）
  └── 例：规则修改、因果逆转

SS 级：半神异能（规则级）
  └── 例：创造规则、概念抹杀

SSS 级：神级异能（权柄级）
  └── 例：规则本源、现实修改
```

### 2.4 诡异侵蚀度

```
侵蚀度范围：0-100%

0-20%：安全（无症状）
21-40%：轻度（偶尔幻觉、失眠）
41-60%：中度（频繁幻觉、身体异化）
61-80%：重度（人格异化、失控倾向）
81-99%：临界（随时可能失控）
100%：异化（变成规则怪物）

降低方式：
- 时间自然恢复：极慢（每天 0.5-1%）
- 安全区休息：中等（每天 2-5%）
- 净化道具：快速但稀有
- 特殊异能者治疗：有效但有人情债
```

### 2.5 详细异能示例

#### 主角异能：**规则之眼**（成长型）

| 阶段 | 能力 | 使用限制 |
|------|------|---------|
| 初始 | 看到简单规则 | 每天 3 次，每次 1 分钟 |
| 进阶 | 看到规则漏洞 | 每天 5 次，每次 3 分钟 |
| 成熟 | 短暂修改规则 | 每天 1 次，持续 10 秒 |
| 完全 | 创造新规则 | 消耗巨大，有反噬 |
| 终极 | 规则本源 | 概念级能力 |

#### 其他异能示例

| 异能名称 | 等级 | 类型 | 能力描述 |
|---------|------|------|---------|
| 死亡预告 | C 级 | 感知系 | 预知 24 小时内死亡危险 |
| 火焰操控 | B 级 | 元素系 | 操控火焰形态和方向 |
| 空间切割 | A 级 | 空间系 | 切割空间造成致命伤害 |
| 因果必中 | S 级 | 因果系 | 攻击必定命中目标 |
| 规则免疫 | S 级 | 规则系 | 对特定规则免疫 |
| 概念抹杀 | SS 级 | 因果系 | 从概念上抹除目标 |

## 六、经典规则区域设计

### 6.1 新手区域（★-★★★）

#### 区域 1：午夜电梯（★★）
```
【规则】
1. 电梯只能按奇数楼层
2. 如果听到女人的笑声，立刻闭眼
3. 电梯到达后，先让里面的人出来再进去
4. 不要在电梯里看镜子

【通关方法】
- 遵守规则即可安全通过
- 主角发现规则漏洞：走楼梯不受限

【奖励】
- 异能觉醒概率提升
- 获得道具"电梯卡"（可短暂免疫空间规则）
```

#### 区域 2：不存在的楼层（★★★）
```
【规则】
1. 这栋楼只有 18 层，没有 19 层
2. 如果电梯显示 19 层，立刻按开门键
3. 不要回应 19 层传来的声音
4. 如果看到穿红衣服的女人，不要对视

【真相】
- 19 层是真实存在的，是实验室
- 规则是为了掩盖真相

【主角破局】
- 利用规则之眼发现规则矛盾
- 主动前往 19 层，发现实验室秘密
```

### 6.2 中级区域（★★★★-★★★★★）

#### 区域 3：七重死亡（★★★★★）
```
【规则】
七条规则，违反任何一条都会死亡：
1. 不能说话
2. 不能回头
3. 不能停步
4. 不能呼吸超过 10 秒
5. 不能心跳超过 100
6. 不能思考"死亡"
7. 不能相信任何人

【破解方法】
- 正常情况几乎无法通过
- 主角发现规则漏洞：规则只说"不能"，没说"必须"
- 可以不说话但写字，不停步但可慢走

【奖励】
- S 级道具"规则碎片"
- 解锁规则之眼进阶能力
```

### 6.3 高级区域（★★★★★★-★★★★★★★）

#### 区域 4：时间循环教室（★★★★★★）
```
【规则】
1. 每天 8 点上课，必须准时到
2. 不能离开教室
3. 不能提及"毕业"
4. 老师说的话都是对的
5. 作业必须完成

【真相】
- 这是一个时间循环区域
- 所有人被困在同一天
- "老师"是规则具现化

【破解方法】
- 主角发现规则第 4 条的漏洞
- 让老师说出一句自相矛盾的话
- 规则崩溃，循环打破

【奖励】
- 时间系异能碎片
- 解锁区域通行权
```
