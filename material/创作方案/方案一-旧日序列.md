# 方案一：《旧日序列》

> **题材**：克苏鲁 + 序列途径 + 都市脑洞
> **定位**：都市异能/克苏鲁/序列进化/群像
> **目标字数**：300-500 万字

---

## 一、核心设定

### 1.1 一句话故事
> 2024 年，月亮变成了一只眼睛。全球 1% 的人觉醒"旧日序列"能力，在疯狂边缘守护人类。

### 1.2 核心冲突
- **表层**：觉醒者 vs 诡异事件
- **里层**：人类 vs 旧日支配者入侵
- **深层**：理智 vs 疯狂（力量代价）

### 1.3 世界观

**表里两层世界**：
```
表层世界（普通人认知）：
- 2024 年，现代社会
- "月瞳现象"：月亮变成眼睛状天文奇观
- 各地出现"异常事件"，官方称其为"集体幻觉"

里层世界（觉醒者认知）：
- 旧日支配者正在苏醒
- 全球 1% 人类觉醒"旧日序列"
- 存在专门处理异常事件的组织
- 每次使用能力都会积累"污染值"
```

**核心规则**：
1. 序列能力源自旧日支配者
2. 使用能力会积累污染值
3. 污染值过高会异化/失控
4. 只有杀死同序列者才能快速净化污染

## 二、序列途径体系

### 2.1 八大途径

基于克苏鲁神话八大权柄设计：

| 途径 | 权柄 | 核心能力 | 失控表现 |
|------|------|---------|---------|
| **窥秘人** | 知识/真相 | 预知、洞察、解析 | 信息过载、大脑异化 |
| **守夜人** | 守护/牺牲 | 防御、净化、封印 | 情感丧失、石化 |
| **梦行者** | 梦境/潜意识 | 入梦、操控、预言 | 现实梦境混淆 |
| **吞噬者** | 吞噬/进化 | 吞噬、模仿、进化 | 永远饥饿、失去人形 |
| **操纵师** | 精神/控制 | 读心、暗示、支配 | 人格分裂、被反噬 |
| **变貌者** | 变形/伪装 | 变身、拟态、融合 | 无法维持人形 |
| **咒言师** | 语言/规则 | 言灵、诅咒、契约 | 语言能力丧失 |
| **送葬人** | 死亡/轮回 | 通灵、复活、送葬 | 被亡者侵蚀 |

### 2.2 序列等级

每条途径分 9 个序列（序列 9 最低，序列 0 最高）：

```
序列 9 → 序列 8 → 序列 7 → 序列 6 → 序列 5
  ↓        ↓        ↓        ↓        ↓
觉醒者   正式    资深    专家    大师
(入门)   (入门)   (中坚)   (精英)   (强者)

序列 4 → 序列 3 → 序列 2 → 序列 1 → 序列 0
  ↓        ↓        ↓        ↓        ↓
半神    神明    高位    顶级    旧日
(质变)  (规则)  (支配)  (近乎    (权柄
                 全能)   化身)
```

### 2.3 详细序列途径（以"窥秘人"为例）

| 序列 | 名称 | 核心能力 | 污染风险 |
|------|------|---------|---------|
| 9 | 观察者 | 超常视力、细节捕捉 | 轻度幻觉 |
| 8 | 解读者 | 解析事物本质、阅读记忆 | 记忆混乱 |
| 7 | 预视者 | 短期预知、危险感知 | 时间感错乱 |
| 6 | 真相师 | 看破伪装、识别谎言 | 无法忍受谎言 |
| 5 | 秘术师 | 禁忌知识、神秘仪式 | 被知识侵蚀 |
| 4 | 先知 | 长期预言、命运观测 | 预见自己死亡 |
| 3 | 全知者 | 近乎全知、信息操控 | 人性丧失 |
| 2 | 真理之眼 | 看穿规则、修改认知 | 成为规则本身 |
| 1 | 洞察之主 | 洞察万物、知晓一切 | 失去自我 |
| 0 | 旧日·求知者 | 权柄化身、全知全能 | 与旧日融合 |

### 2.4 污染值系统

```
污染值范围：0-100%

0-30%：安全区（无明显症状）
31-50%：警告区（轻度幻觉、失眠）
51-70%：危险区（严重幻觉、身体异化）
71-90%：临界区（人格异化、失控倾向）
91-100%：失控区（完全异化、成为怪物）

净化方式：
- 自然恢复：每天降低 1-5%（极慢）
- 净化仪式：需要特殊材料和场所
- 杀死同序列：快速降低但有道德代价
- 特殊道具：一次性净化物品
```

## 三、主角设定

### 3.1 基本信息

| 项目 | 设定 |
|------|------|
| **姓名** | 陆离 |
| **年龄** | 24 岁 |
| **身份** | 前刑警→异常事件调查员 |
| **序列途径** | 窥秘人→全知者→旧日 |
| **金手指** | 污染值锁定（使用能力不增加污染） |
| **性格** | 冷静、理性、有底线但不圣母 |
| **缺陷** | 无法完全信任他人（职业后遗症） |

### 3.2 核心欲望
- **表层**：调查妹妹失踪真相
- **深层**：找到让妹妹回来的方法
- **终极**：在疯狂世界中保持人性

### 3.3 核心缺陷
- 过度理性，习惯把人当线索而非人
- 对"真相"有病态执念
- 害怕自己终有一天会失控

## 五、主线剧情大纲

### 5.1 整体架构

```
第一卷：月瞳降临（1-100 章）
├── 第 1-10 章：月瞳现象，妹妹失踪
├── 第 11-30 章：觉醒序列 9，加入异常事务局
├── 第 31-50 章：调查第一起异常事件
├── 第 51-70 章：发现妹妹线索，序列晋升
├── 第 71-100 章：揭开第一个旧日降临阴谋

第二卷：序列战争（101-250 章）
├── 第 101-130 章：序列者之间的冲突
├── 第 131-170 章：真理之眼的邀请
├── 第 171-210 章：旧日教会的阴谋
├── 第 211-250 章：序列 5 晋升，成为专家

第三卷：旧日苏醒（251-400 章）
├── 第 251-280 章：第一个旧日完全苏醒
├── 第 281-320 章：全球危机，联合对抗
├── 第 321-360 章：妹妹真相一角
├── 第 361-400 章：序列 3 晋升，半神级别

第四卷：真相与牺牲（401-550 章）
├── 第 401-430 章：找到妹妹，但已成"容器"
├── 第 431-470 章：拯救妹妹的方法
├── 第 471-510 章：最终决战准备
├── 第 511-550 章：牺牲与选择

第五卷：新秩序（551 章 - 完结）
├── 第 551-580 章：旧日之战
├── 第 581-600 章：最终抉择
└── 结局：守护/毁灭/新生
```

### 5.2 核心主线

**明线**：陆离调查妹妹失踪→发现旧日阴谋→阻止旧日降临

**暗线**：
- 月瞳现象真相（旧日苏醒前兆）
- 序列能力来源（旧日权柄碎片）
- 污染值本质（旧日侵蚀程度）
- 陆离金手指来源（特殊身份）

**终极悬念**：
- 妹妹为何被选中成为"容器"
- 陆离为何能免疫污染
- 旧日苏醒的真正目的

### 5.3 关键剧情节点

| 章节 | 事件 | 意义 |
|------|------|------|
| 第 1 章 | 月瞳降临，妹妹失踪 | 故事起点 |
| 第 10 章 | 序列觉醒，金手指发现 | 能力觉醒 |
| 第 30 章 | 加入异常事务局 | 获得组织 |
| 第 50 章 | 首次失控边缘 | 污染代价 |
| 第 100 章 | 第一个旧日降临被阻止 | 第一阶段胜利 |
| 第 200 章 | 序列 5 晋升 | 成为专家 |
| 第 300 章 | 妹妹真相揭露 | 情感高潮 |
| 第 400 章 | 序列 3 半神 | 实力质变 |
| 第 500 章 | 最终抉择 | 主题升华 |
| 完结 | 新秩序建立 | 故事收尾 |

## 六、核心爽点设计

### 6.1 爽点类型

| 爽点类型 | 具体表现 | 出现频率 |
|---------|---------|---------|
| **序列晋升** | 获得新能力、地位提升 | 每 30-50 章 |
| **信息差** | 主角知道别人不知道的秘密 | 持续 |
| **污染免疫** | 别人不敢用的能力主角随便用 | 持续 |
| **智斗破局** | 用推理和知识战胜敌人 | 每 10-20 章 |
| **身份掉马** | 隐藏身份暴露时的震惊 | 每 50 章 |
| **守护成功** | 保护重要的人 | 情感高潮 |
| **反转** | 看似必死局面反转 | 每卷高潮 |

### 6.2 钩子设计

**章末钩子模板**：
```
- "他不知道的是，……"
- "下一秒，他看到了……"
- "电话那头，是妹妹的声音"
- "序列 0 的真相，是……"
```

**卷末钩子模板**：
```
- 重大真相揭露
- 重要角色死亡
- 主角身份曝光
- 新的危机出现
```

## 七、开篇设计（前 10 章详细）

### 第 1 章：月瞳
```
开场：2024 年 9 月 1 日，陆离和妹妹陆瑶一起看月亮
异常：月亮变成眼睛形状，全球直播
异变：妹妹突然昏迷，留下诡异符号
悬念：符号与某种古老存在有关
章末钩子："那不是符号，是……一只眼睛的图案"
```

### 第 2 章：失踪
```
妹妹醒来后行为异常
陆离发现妹妹在画同一个图案
妹妹再次昏迷，醒来后说"我不是你妹妹"
陆离报警，但警察查不出异常
章末钩子："监控里，妹妹昨晚从未离开过房间"
```

### 第 3 章：觉醒
```
陆离开始调查，发现自己能看到别人看不到的"污染痕迹"
遇到第一个异常事件：某大楼集体幻觉
在事件中，陆离觉醒了序列能力
章末钩子："序列 9 观察者，欢迎来到真实的世界"
```

### 第 4-10 章大纲
```
第 4 章：接触异常事务局
第 5 章：第一次任务
第 6 章：发现污染值系统
第 7 章：金手指觉醒（污染值锁定）
第 8 章：序列能力实战
第 9 章：发现妹妹线索
第 10 章：正式加入异常事务局
```

## 八、核心主题
