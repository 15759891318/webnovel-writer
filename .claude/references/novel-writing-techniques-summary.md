### 1.8 节奏把控技巧

**已有文档**：
- `.claude/skills/webnovel-write/references/writing/pacing-control.md`
- `.claude/agents/pacing-checker.md`

**节奏层次**：
```
宏观节奏（全书）→ 中观节奏（每卷）→ 微观节奏（每章）
```

**各卷节奏设计**：
- 第一卷：慢→中→快（世界观引入）
- 第二卷：中→快→中（势力扩张）
- 第三卷：快→最快→中（真相一角）
- 第四卷：最快→爆发→收尾（最终决战）

### 1.10 配角塑造技巧

**已有文档**：
- `.claude/skills/webnovel-write/references/writing/supporting-characters.md`

**配角设计五要素**：
1. 言行举止（能否通过对话认出角色）
2. 性格能力（是否有辨识度）
3. 背景故事（是否有独立于主角的过去）
4. 动机目标（行为是否有内在驱动力）
5. 正邪立场（是否有自己的判断）

**典型案例**：
- 邓恩·史密斯（《诡秘之主》）：导师型配角，牺牲场景震撼
- 乔家劲（《十日终焉》）：忠义型配角，行动派

### 1.11 世界观引入技巧

**已有文档**：
- `.claude/skills/webnovel-write/references/writing/world-building.md`

**核心原则**：
- 故事承载（设定通过剧情展现）
- 分层释放（每章只透露 20%）
- 读者探索（让读者有"发现"快感）

**展示节奏**：
```
第 1 章：5% → 第 3 章：20% → 第 10 章：40% → 第 30 章：60% → 第 50 章：80% → 第 100 章：100%
```

---

### 1.12 世界观展示技巧

**已有文档**：
- `.claude/skills/webnovel-write/references/writing/world-exposure.md`

**三大禁忌**：
1. 信息倾倒（前 3 章塞入大量设定）
2. 说明文式（像教科书而非故事）
3. 主角自说自话（知道自己不该知道的信息）

**展示方法**：
- 通过对话展示（问答形式）
- 通过冲突展示（实力对比）
- 通过探索展示（主角发现）

## 三、技巧使用建议

### 3.1 新手推荐顺序

1. **首先掌握**：开篇技巧 + 去 AI 味技巧
2. **其次掌握**：对话技巧 + 人物塑造
3. **进阶掌握**：战斗技巧 + 爽点设计
4. **高级掌握**：节奏把控 + 世界观展示

## 四、技巧检查清单

### 4.1 章节写完后检查

**基础检查**：
- [ ] 无 AI 味句式（模式化词汇/过度连接）
- [ ] 对话无"他说/她说"单调循环
- [ ] 角色言行一致（无 OOC）
- [ ] 情节连贯（无逻辑漏洞）

**进阶检查**：
- [ ] 本章有爽点或同等兑现
- [ ] 章末有钩子（驱动点下一章）
- [ ] 节奏张弛有度（快慢变化）
- [ ] 世界观分层释放（不过密）

**高级检查**：
- [ ] 配角有独立动机（非工具人）
- [ ] 战斗有三回合结构
- [ ] 情感有铺垫和释放
- [ ] 悬念有埋有收
