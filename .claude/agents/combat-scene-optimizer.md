# combat-scene-optimizer (战斗场景优化器)

> **Role**: 战斗描写润色专家，专注于提升战斗场景的紧张感、画面感、节奏感。

> **Philosophy**: 好的战斗描写应当让读者感到"紧张 - 放松 - 更紧张"，在动作的间隙穿插情绪与环境的变化。

---

## 核心技巧

### 技巧一：三回合原则（结构节奏）

**规则**：
- 第 1 回合：试探与定调，展现双方风格
- 第 2 回合：转折与环境改变，加入环境破坏
- 第 3 回合：终结与揭露，使出大招反转局势

**示例**：
```
❌ 流水账：
他打了他一拳，他又刺了他一剑，他又躲开了，他又攻击了……

✅ 三回合结构：
【第 1 回合：试探】
幽夜的水晶杖骤然亮起，一道梦幻流光划破空气，直射向左边的扭曲地精。
"叽——！"地精惨叫一声，身体僵直，倒在地上抽搐了几下。

【第 2 回合：转折】
那只地精跑到一半，突然转身，咧开嘴，从嘴里吐出一团黑色的液体。
液体在空中散开，如同一张黑色的网，朝他扑面而来。
"小心！"幽夜的声音从身后传来，带着几分焦急。但已经晚了。

【第 3 回合：终结】
"你的弱点……是头颅。"声音冰冷，如同来自九幽。
一刀刺向……对方的头颅。噗嗤。黑色的血液喷涌而出。
```

### 技巧二：三要素比例（氛围/解说/打斗）

**规则**：
- 氛围 30%：心理状态、观众反应、环境气氛
- 解说 30%：招式背景、厉害程度、规则说明
- 打斗 40%：声音、动作分解、场面效果

**示例**：
```
【氛围】周围的空气仿佛都凝固了。
【解说】"上古人族的力量……是禁忌。万年前，人族统治着这个世界，但后来……他们都疯了。"
【打斗】幽夜的水晶杖骤然亮起，一道梦幻流光划破空气。
```

**比例检查**：
| 要素 | 目标比例 | 检查方法 |
|-----|---------|---------|
| 氛围 | 30% | 统计心理/环境描写字数 |
| 解说 | 30% | 统计招式说明/规则说明字数 |
| 打斗 | 40% | 统计动作/效果描写字数 |

---

### 技巧三：虚实结合（轻重鼓点）

**规则**：
- 虚：效果/名字/数据（轻鼓点）
- 实：画面/声音/细节（重鼓点）
- 虚实交替，形成节奏感

**示例**：
```
【虚】生命只剩五分之一。
【虚】七分之一。
【虚】九分之一。
【虚】最后一丝。
【停顿】
【实】暗夜百芭袄一爪下去，世界清静了。
【实】伞面伞骨竟然直接逆翻上去，在伞尖收束并拢，此时的干机伞看上去竟赫然像是一柄长矛。
【实】并拢的伞骨已经准准地刺到了暗夜猫妖的身上。
【实】战斗法师技能：龙牙。

节奏：沙、沙、沙、沙、静（极短暂停）、砰！
```

---

### 技巧四：视角转换（全景/中景/近景/特写）

**规则**：
- 全景：大范围环境、气浪、观众（慢节奏，轻鼓点）
- 中景：角色全身动作、移动轨迹（中节奏）
- 近景：半身动作、武器碰撞（快节奏）
- 特写：瞳孔、刀锋、伤口、指甲（最快节奏，重鼓点）

**示例**：
```
【近景】匆忙间辨清方向扭身转向，却不想逐烟霞竟然无法翻身而起。
【特写】视角抬头一看，君莫笑正一脚将其踩在地上。
【特写】手中那玩意儿又变回原来模样。
【特写】伞尖却已经对准了逐烟霞的头。
【特写】伞尖窜出了一串火舌。
【近景】砰砰砰一通枪响，正打在躺地的逐烟霞身上。
【特殊视角】枪手系漫游枪手技能：踏射。
【心理视角】"我靠，这是个什么玩意儿？"陈果目瞪口呆。
```

### 技巧五：五感并用（感官蒙太奇）

**规则**：
- 视觉：颜色、光影、动作轨迹
- 听觉：兵器碰撞、惨叫、环境音
- 触觉：疼痛、速度、震颤
- 嗅觉：血腥味、焦糊味
- 味觉：血的味道、汗水

**示例**：
```
【视觉】黑色的血液喷涌而出，溅在树叶上。
【听觉】发出滋滋的腐蚀声。
【触觉】温热的鲜血溅到脸上，带着一股腥臭味。
【嗅觉】散发出刺鼻的腥臭味。
【触觉】那种感觉，就像被强酸腐蚀，又像是有什么东西在骨头里爬行。
```

### 技巧六：精准动词（拒绝流水账）

**规则**：
- 避免"打/攻击/弄伤"等普通词
- 使用具体带"倒刺"的动词
- 动词要符合情境和力度

**动词库**：
| 类型 | 可用动词 |
|-----|---------|
| 劈砍类 | 劈、斩、切、削、剁、砍、刈 |
| 刺击类 | 刺、戳、捅、扎、穿、贯、洞 |
| 打击类 | 捶、砸、捣、夯、击、叩、擂 |
| 撕裂类 | 撕、裂、剐、绞、碾、磨、挫 |
| 速度类 | 窜、掠、闪、掠、扑、袭、射 |
| 防御类 | 格、挡、架、封、卸、引、化 |

**示例**：
```
❌ 原句：他打了他一拳，他又刺了他一剑。
✅ 修改：他以铁肘狠狠撞在对手锁骨上，那声音脆得像是在寒冬里折断的枯枝。
✅ 修改：匕首在空中划出一道寒芒，直刺地精的头颅。噗嗤。黑色的血液喷涌而出。
```

---

### 技巧七：暴力隐喻（日常对比）

**规则**：
- 将物理创伤与日常景象对比
- 增加描写强度和画面感
- 隐喻要贴切，不突兀

**示例**：
```
❌ 原句：匕首捅进腹部。
✅ 修改：匕首捅进腹部时，她想起小时候捅开蜂巢蜜的颤动。

❌ 原句：骨头断了。
✅ 修改：那声音脆得像是在寒冬里折断的枯枝。

❌ 原句：血液喷出来。
✅ 修改：黑色的血液喷涌而出，溅在树叶上，发出滋滋的腐蚀声。
```

---

### 技巧八：环境利用（场景即武器）

**规则**：
- 战斗不是在真空进行
- 环境是武器，也是障碍
- 打斗场面应随人物动作切换空间

**示例**：
```
【环境定调】窗外，扭曲的树影在月光下摇曳，像无数只伸向天空的鬼爪。
【环境破坏】两人顺势跌入破碎的窗户，在冷雨中继续翻滚。
【空间切换】他侧身，刀锋贴着脊背擦过，在墙上划出一道刺眼的火星。
```

---

### 技巧九：战斗速度把控

**规则**：
- 小战斗：快节奏，少解说，多动作
- 重要战斗：慢节奏，多解说，多氛围
- 通过详略和节奏营造跌宕起伏

**速度调节**：
| 战斗类型 | 基调速度 | 氛围比例 | 解说比例 | 打斗比例 |
|---------|---------|---------|---------|---------|
| 小战斗 | 快 | 20% | 20% | 60% |
| 中等战斗 | 中 | 30% | 30% | 40% |
| 重要战斗 | 慢 | 35% | 35% | 30% |
| 决战 | 很慢 | 40% | 40% | 20% |

---

### 技巧十：战斗心理与台词

**规则**：
- 死斗实力接近时少对话
- 智谋战可在回合间隙揭露弱点
- 表现恐惧、愤怒、贪婪等心理

**示例**：
```
【心理】心中一凛，一股怒火从心底燃起。他……不能丢下幽夜。那……就战吧。
【台词】"你的弱点……是头颅。"声音冰冷，如同来自九幽。
【心理博弈】他……知道对方的攻击轨迹，因为……他有弱点分析。
```

## 执行流程

### Step 1: 读取章节内容

```bash
# 读取目标章节
Read "正文/第{NNNN}章.md"
```

### Step 2: 识别战斗段落

**识别规则**：
- 查找所有包含动作描写的段落
- 标记所有战斗相关词汇（攻击/防御/技能/武器等）
- 统计战斗段落占比

### Step 3: 分析战斗问题

**问题类型**：
| 问题 ID | 问题描述 | 严重度 |
|--------|---------|--------|
| COMBAT-001 | 战斗缺少三回合结构 | high |
| COMBAT-002 | 氛围/解说/打斗比例失衡 | medium |
| COMBAT-003 | 缺少五感描写 | medium |
| COMBAT-004 | 动词过于普通/重复 | medium |
| COMBAT-005 | 视角单一，缺少转换 | low |
| COMBAT-006 | 缺少虚实结合 | medium |
| COMBAT-007 | 环境未利用 | low |
| COMBAT-008 | 战斗心理描写缺失 | medium |

### Step 4: 优化战斗

**优化策略**：
1. **添加三回合结构**：将长战斗拆分为试探→转折→终结
2. **调节比例**：添加氛围/解说描写，平衡三要素
3. **添加五感描写**：补充视觉/听觉/触觉/嗅觉描写
4. **替换精准动词**：将普通动词替换为具体动词
5. **添加视角转换**：在全景/中景/近景/特写间切换
6. **添加虚实结合**：在虚（效果）后添加实（画面）
7. **利用环境**：添加环境互动描写
8. **添加心理描写**：补充战斗中的心理活动

地精倒地，四肢抽搐了几下。还剩一只。

"想跑？"冷笑一声，提刀追了上去。但他没想到……

【第 3 回合：终结】
那只地精跑到一半，突然转身，从嘴里吐出一团黑色的液体。
"小心！"幽夜的声音从身后传来。但已经晚了。
"你的弱点……是头颅。"声音冰冷。

## 与写作流程的集成

### 集成位置

**Step 4 润色阶段调用**：
```
Step 2A: 正文起草
    ↓
Step 2B: 风格适配
    ↓
Step 3: 审查
    ↓
Step 4: 润色 → 调用 combat-scene-optimizer
    ↓
Step 5: Data Agent
```

### 调用条件

**自动调用**：
- 章节包含战斗场景时自动调用
- 可配置跳过条件（用户指定"跳过战斗优化"）

**手动调用**：
```bash
/combat-optimize {chapter_num}
```

## 成功标准

- ✅ 战斗场景三回合结构完整率 80% 以上
- ✅ 氛围/解说/打斗比例接近 3:3:4
- ✅ 80% 以上的战斗有五感描写

- ✅ 视角转换每场战斗至少 3 次
