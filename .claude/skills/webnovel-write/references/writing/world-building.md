# 世界观引入技巧 (World-Building)

> 基于 6 本热门小说分析提炼的世界观写作技巧
> 来源：《诡秘之主》《十日终焉》《斩神》《序列：吃神者》《我不是戏神》《诡舍》

---

## 一、世界观层次

### 1.1 表里两层世界

**所有 6 本小说的共同特点**：

```
表层世界（普通人认知）
├── 现代都市/正常社会
├── 异常事件被掩盖
└── "那是集体幻觉"

里层世界（觉醒者认知）
├── 神明/诡异/序列存在
├── 专门处理异常的组织
└── 使用能力有代价
```

**示例对比**：

| 作品 | 表层 | 里层 |
|------|------|------|
| 《诡秘之主》 | 维多利亚异界 | 克苏鲁 + 序列途径 |
| 《斩神》 | 现代都市 | 神明入侵 + 守夜人 |
| 《序列》 | 安全区日常 | 神明威胁 + 序列者 |
| 《十日》 | 封闭游戏空间 | 轮回真相 + 生肖体系 |

### 1.2 世界观揭示节奏

```
第 1-10 章：核心设定（20%）
├── 金手指觉醒
├── 主角处境
└── 第一次接触异常

第 11-30 章：世界观基础（40%）
├── 能力体系介绍
├── 社会结构
└── 加入组织

第 31-50 章：势力分布（60%）
├── 主要势力介绍
├── 敌人出现
└── 利益冲突

第 51-100 章：深层秘密（80%）
├── 世界真相一角
├── 主线阴谋
└── 更大格局

第 101 章+：完整世界（100%）
├── 多地图展开
├── 完整历史
└── 终极真相
```

## 二、世界观引入方式

### 2.1 对话/课堂引入

**代表作品**：《序列：吃神者》《斩神》

**示例 1（历史课）**：
```
"2000 年，太阳消失了。"
历史老师在讲台上说。
"世界进入永夜，人类被迫迁入安全区。"
"从那以后，出现了两种人。"
"普通人，和越界者。"
秦思洋坐在教室后排，听着这些他已经知道的历史。
但他知道的，比老师说的多得多。
——比如，越界者是怎么来的。
——比如，神明是可以被杀死的。
```

**示例 2（医生问诊）**：
```
"你什么时候开始看到那些东西的？"医生问。
"十岁。"林七夜说。
"看到什么？"
"月亮上的眼睛。"
医生在病历上写着什么。
"那之后呢？"
"我失明了。"
"但现在你能'看到'，对吗？"
"嗯，闭眼也能看到周围十米。"
医生停下笔，看着他。
"这不是病，这是……天赋。"
```

**优点**：
- 自然不突兀
- 读者和主角同步了解
- 可以穿插主角内心想法

### 2.2 规则/系统引入

**代表作品**：《十日终焉》《诡舍》

**示例（游戏规则）**：
```
【终焉之地规则】
1. 每十日为一个轮回
2. 轮回结束时，所有人死亡
3. 部分人可以保留记忆重生
4. 只有逃离终焉之地，才能打破轮回

"记住了吗？"山羊头问。
"这就是你们要面对的世界。"
"要么逃，要么死。"
"没有第三条路。"
```

**优点**：
- 快速建立紧张感
- 规则清晰明确
- 读者容易理解

### 2.3 行动展示引入

**代表作品**：《我不是戏神》

**示例**：
```
陈伶深吸一口气，走上舞台。
聚光灯打在他身上。
【戏神系统激活】
【当前角色：小生】
【完成度：0%】
【演技越好，能力越强】

他开始了表演。
十分钟后，演出结束。
【完成度：35%】
【获得能力：身段（精通）】

陈伶握了握拳，感受着新获得的力量。
——这就是戏神的能力。
——这就是他在这个世界生存的资本。
```

**优点**：
- 节奏快
- 直观展示
- 不拖沓

### 2.4 文档/日记引入

**代表作品**：《诡秘之主》

**示例（日记）**：
```
【三月十五日 晴】
今天我加入了值夜者，成为了一名非凡者。
队长说，这条路上没有回头路。
我……回不了头了。

【三月二十日 阴】
第一次执行任务。
原来这个世界上真的有怪物。
原来序列途径是真的存在。
我想回家，但我回不去了。

【四月一日 雨】
晋升序列 8 了。
扮演法是消化的关键。
我要扮演……一个小丑。
真是讽刺。
```

**优点**：
- 信息密度高
- 可以跳跃时间
- 增加真实感

## 三、能力体系设计

### 3.1 能力体系要素

```
完整的能力体系包含：

1. 能力名称（序列/异能/途径）
2. 等级划分（9→0/F→SSS）
3. 获取方式（魔药/觉醒/任务）
4. 使用限制（消耗/冷却/代价）
5. 成长路径（如何变强）
```

### 3.2 能力体系示例

**示例 1：序列途径（《诡秘之主》风格）**
```
22 条神之途径，每条 10 个序列

占卜家途径：
序列 9：占卜家
  能力：占卜、灵性感知
  扮演守则：给人占卜、传播神秘学

序列 8：小丑
  能力：身体控制、表情伪装
  扮演守则：做滑稽表演、逗人笑

序列 7：魔术师
  能力：火焰操控、空间戏法
  ……

序列 0：愚者
  能力：操控命运、时空
```

**示例 2：异能等级（都市异能风格）**
```
等级划分：F→E→D→C→B→A→S→SS→SSS

F 级：基础异能（微弱效果）
E 级：普通异能（实用效果）
D 级：精良异能（战斗可用）
C 级：精英异能（战术价值）
B 级：大师异能（战略价值）
A 级：宗师异能（城市级）
S 级：传奇异能（文明级）
SS 级：半神异能（规则级）
SSS 级：神级异能（权柄级）
```

### 3.3 代价系统设计

**有代价的能力体系更真实**：

| 代价类型 | 示例 | 效果 |
|---------|------|------|
| 污染值 | 每次使用增加污染 | 过高会失控 |
| 理智值 | 克苏鲁 SAN 值 | 过低会疯狂 |
| 寿命 | 使用能力减寿 | 慎重使用 |
| 记忆 | 失去部分记忆 | 情感代价 |
| 身体 | 身体异化 | 可见代价 |

**示例（污染值）**：
```
【污染值：35%】
【状态：警告区】
【症状：轻度幻觉、失眠】
【建议：尽快净化】

"又增加了。"克莱恩皱眉。
每次使用序列能力，污染值都会上升。
达到 100% 就会失控。
他必须找到净化的方法。
```

## 四、势力设计

### 4.1 势力类型

| 势力类型 | 功能 | 代表 |
|---------|------|------|
| 官方组织 | 提供资源、发布任务 | 异常事务局、守夜人 |
| 学院 | 培养新人 | 守夜人学院、南荣大学 |
| 隐秘组织 | 信息共享、特殊资源 | 塔罗会、戏道古藏 |
| 敌对组织 | 制造冲突 | 旧日教会、清道夫 |
| 中立组织 | 提供服务 | 深渊商会、拍卖行 |

### 4.2 势力关系设计

```
经典三角关系：

        官方组织（友好）
           /    \
          /      \
    主角团队 ———— 隐秘组织（中立）
          |
          |
      敌对组织（敌对）

作用：
- 官方组织：提供合法身份和资源
- 隐秘组织：提供特殊信息和逼格
- 敌对组织：制造冲突和压力
```

### 4.3 势力信息卡

**为每个主要势力设计信息卡**：
```
【异常事务局】
性质：官方组织
立场：友好
规模：全国 3 个分局，500+ 人员
资源：序列药剂、封印物、情报网
代表人物：局长陈建国（序列 3）
与主角关系：雇主/靠山

【旧日教会】
性质：邪神组织
立场：敌对
规模：未知，分散各地
资源：禁忌知识、信徒、仪式
代表人物：教主"神仆"（序列 2）
与主角关系：敌人
```

## 五、世界观细节填充

### 5.1 货币体系

```
设计货币兑换关系，增加真实感

示例（诡秘之主）：
1 金镑 = 20 苏勒
1 苏勒 = 12 便士

示例（原创）：
1 晶币 = 100 积分
1 积分 = 10 贡献点

主角购买物品时，写明价格
让读者对货币有概念
```

### 5.2 地理设定

```
设计主要地图，增加世界感

示例：
【安全区】
- 联合政府所在地
- 普通人 + 序列者混居
- 相对安全

【荒野】
- 神明游荡区域
- 危险但资源丰富
- 序列者狩猎场

【禁区】
- 高等神明巢穴
- 序列 5 以下禁止进入
- 有大量遗迹
```

### 5.3 历史设定

```
设计世界历史，增加深度

示例时间线：
2000 年：太阳消失，永夜开始
2005 年：第一批越界者出生
2010 年：安全区建立
2020 年：序列体系发现
2024 年：主角登场
```
