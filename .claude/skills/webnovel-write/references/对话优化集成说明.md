## 对话优化六大技巧

### 技巧一：用动作描写代替"他说"

```
❌ 原句："这个我不能同意。"他说。
✅ 修改：他紧锁眉头，狠狠地拍了一下桌子，"这个我不能同意。"
```

### 技巧二：用神态/情绪描写补充对话

```
❌ 原句："你为什么还没走？"她说。
✅ 修改：她盯着他的眼睛，眼角带着一丝不屑，"你为什么还没走？"
```

### 技巧三：用心理活动或环境描写自然衔接

```
❌ 原句："今天下雨了。"他说。
✅ 修改："今天下雨了。"他看着窗外连绵的阴雨，发出一声叹息。
```

### 技巧四：省略提示语（跳出对话框）

```
❌ 原句：
"你要去哪？"他问。
"火车站。"她答。
"我就知道你还要走。"他说。

✅ 修改：
"你要去哪？"
"火车站。"
"我就知道你还要走。"
```

### 技巧五：使用多元化的动词描述

| 情绪类型 | 可用动词 |
|---------|---------|
| 愤怒 | 咆哮、怒吼、喝斥、质问 |
| 悲伤 | 哽咽、低泣、呢喃、叹息 |
| 高兴 | 欢笑、雀跃、欢呼、笑道 |
| 恐惧 | 惊呼、颤抖、结巴、低语 |
| 轻蔑 | 冷笑、嘲讽、嗤笑、不屑 |
| 严肃 | 正色、沉声、郑重、告诫 |
| 温柔 | 柔声、轻抚、温言、安抚 |

### 技巧六：中式/西式/混搭式对白

**中式**（对白在句尾）：
```
他皱了皱眉，有些不悦地说道：「这个我不能同意。」
```

**西式**（对白在句首）：
```
「这个我不能同意。」他皱了皱眉，有些不悦地说道。
```

**混搭**（对白被叙述打断）：
```
「这个，」他顿了顿，似乎在斟酌用词，「我不能同意。」
```

## 集成到写作流程

### 集成位置

**Step 2B 后 / Step 3 前**：
```
Step 2A: 正文起草
    ↓
Step 2B: 风格适配
    ↓
Step 2C: 对话优化 ← 新增
    ↓
Step 3: 审查
```

### 调用方式

**自动调用**（标准模式）：
- 每章写作完成后自动调用 `dialogue-optimizer` agent
- 优化完成后进入 Step 3 审查

**手动调用**：
```bash
/Dialogue-optimize {chapter_num}
```

**跳过条件**：
- `/webnovel-write --fast`：跳过
- `/webnovel-write --minimal`：跳过
- 用户明确指定"跳过对话优化"

## 优化标准

### 优化目标

| 指标 | 目标值 |
|------|-------|
| 单调"他说/她说"使用率 | 降低 70% 以上 |
| 有动作/神态描写的对话 | 80% 以上 |
| 两人对话省略提示语比例 | 50% 以上 |
| 对话动词多元化 | 不重复使用同一动词 |

### 检查清单

| 问题 | 检查方法 | 优化建议 |
|------|---------|---------|
| 单调提示语 | 连续 3 次以上"他说/她说" | 替换为动作/神态描写 |
| 缺乏动作 | 对话无任何动作描写 | 添加符合情境的动作 |
| 缺乏神态 | 对话无神情描写 | 添加符合情绪的神态 |
| 提示语冗余 | 两人对话频繁使用提示语 | 省略不必要的提示语 |
| 动词单一 | 通篇只用"说" | 使用多元化动词 |
| 对话与人物不符 | 对话不符合人物性格 | 调整对话风格 |

## 优化示例

### 示例一：连续"他说"优化

**优化前**：
```
"你来了。"他说。
"嗯。"她说。
"坐吧。"他说。
"谢谢。"她说。
"最近怎么样？"他说。
"还好。"她说。
```

**优化后**：
```
"你来了。"他抬起头，放下手中的文件。
"嗯。"她轻轻关上门，走到他对面的椅子前。
"坐吧。"他指了指椅子，嘴角带着一丝笑意。
"谢谢。"她坐下，双手交叠放在膝上。
"最近怎么样？"他身体前倾，目光关切。
"还好。"她避开他的目光，声音有些低沉。
```

---

### 示例二：添加动作描写

**优化前**：
```
"我不会原谅你的。"她说。
"我知道。"他说。
"但我还是要试试。"她说。
```

**优化后**：
```
"我不会原谅你的。"她转过身，泪水在眼眶里打转。
"我知道。"他低下头，双手紧握成拳。
"但我还是要试试。"她深吸一口气，声音坚定。
```

---

### 示例三：省略冗余提示语

**优化前**：
```
"你要去哪？"他问。
"火车站。"她答。
"我就知道你还要走。"他说。
"那你为什么还要拦我？"她问。
"因为我爱你。"他说。
```

**优化后**：
```
"你要去哪？"
"火车站。"
"我就知道你还要走。"
"那你为什么还要拦我？"
"因为我爱你。"
```

## 常见问题解答

### Q1: 对话优化会不会改变剧情？

**A**: 不会。对话优化只改变对话的表达方式，不改变剧情内容、事实、人物关系。

### Q2: 优化后对话会不会太啰嗦？

**A**: 不会。好的对话优化应该简洁有力，一句话即可，如"他点点头"、"她笑了笑"。

### Q3: 如何保留作者原有风格？

**A**: 设置 `preserve_author_style: true`，优化时会保留作者原有的对话风格。

### Q4: 优化失败怎么办？

**A**: 优化失败不会阻断流程，会记录失败原因并跳过该章节的对话优化。
