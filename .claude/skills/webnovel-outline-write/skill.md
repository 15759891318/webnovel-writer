# 细纲驱动章节写作 (Outline-Driven Writing)

## 目标

- 严格按照细纲生成章节内容，不发散、不添加未设定内容
- 每段对应细纲中的具体剧情点
- 主视角切换时明确标注
- 自动执行写作风格检测，确保合规

---

## 细纲格式

### 标准细纲结构

```markdown
# 第{NNNN}章 细纲

## 章节信息
- 章号：{NNNN}
- 标题：{章节标题}
- 字数目标：{2300-3000}
- 主视角：{角色名}

## 剧情节拍

1. XX 回到 XX（此处适当描写场景）
2. 众人在 XX 讨论
3. XX 意识到 XX
4. XX 开始调查 XX（通过 XX）
5. XX 发现 XX
6. XX 质问 XX（此处渲染 XX 情绪）

---
**截止于此完结**
```

---

## 执行原则

### 1. 严格遵循细纲

- **禁止发散**：不允许描写任何世界书及前文中未提及的机制或设定
- **禁止添加**：不添加细纲外的情节、角色、设定
- **禁止解释**：不对角色行为做解释性说明

### 2. 视角切换规则

- 主视角切换时必须明确标注："主视角切回到 XX"
- 单章节主视角切换不超过 2 次
- 视角切换应当在段落之间，不在段落内部

### 3. 场景描写原则

- 不需要大段描写场景
- 将故事重点放在人物对话、剧情推进、角色塑造上
- 场景描写服务于剧情，不单独存在

---

## 写作要求（强制）

### 禁止句式

**绝对避免**：
- "这句话像是投入平静湖面的巨石"类似句型

**强制禁止比喻词语**：
- 火山、心湖、油锅、石子、湖面、针刺、羽毛、重锤
- 掀起、巨浪、泛起、涟漪、惊雷、某种、水中、波澜
- 水滴、炸弹

**强制禁止情绪暗示**：
- "眼神/眼底闪过/闪烁{情绪/态度}"
- "带着 xx 意味"、"带着某种"、"不再是……而是带着……"
- "带着一种"、"并没有立刻"

**强制禁止惰性情绪状态描述**：
- "声音/表情/语气平淡或平静，没有太多情绪"
- "近乎情绪状态"、"充满了某种"

**强制禁止生理/动作细节**：
- "指节描写与动作"、"嘴角弧度"、"审视"

**强制禁止状态描述**：
- "不容拒绝"、"不易察觉"、"难以察觉"、"不可置信"、"不容置疑"

**强制禁止动物比喻**：
- "像一只 XX"

**强制禁止平衡句式**：
- "不是…而是…"、"不仅…更是…"
- 如需表达转折，把一段话分成两段说

---

### 写作规范

**客观化、中立化**：
- 不对事物做任何评价与解释性描述
- 不主观判断或推测结果

**心理描写**：
- 不应与剧情中已提到的内容重复

**展示而非讲述**：
- 无需给出结论与说明的"讲述"式描写
- 直接陈述事实，不做多余解释
- 避免任何形容词与副词、各类修辞的修饰
- 拒绝对人物声音、动作做出任何评价与描述

**尊重读者智力**：
- 不对角色言行神态背后的动机或内涵做任何解释说明
- 相信读者能够自行构建意义

**略去总结性名词化表达**：
- 不写"他……。这个动作打破了……"
- 直接写"他……，打破了……"

**创作重心**：
- 不在人物的神态（尤其避免眼神的描写）
- 无需展开叙述

**叙述真实感**：
- 让叙述可以以假乱真，形成不输于历史叙述的真实感

**群体差异化**：
- 塑造不同群体多样化的价值观念，行为举止，言谈差异

**介词连词副词**：
- 少用：的、虽然、但是、因为、所以、而、也

---

## 执行流程

### Step 0：预检

**必须校验**：
1. 细纲文件存在：`大纲/第{NNNN}章 -细纲.md`
2. 世界书/设定集可用
3. 前文最新章节可用

**缺失处理**：
- 缺失细纲 → 阻断并提示用户先创建细纲
- 缺失设定 → 阻断并提示用户补充设定

### Step 1：读取细纲与世界书

**读取内容**：
```bash
# 读取细纲
Read "大纲/第{NNNN}章 -细纲.md"

# 读取世界书/设定集（按需）
Read "设定集/{相关设定}.md"

# 读取前文（承接上章）
Read "正文/第{NNNN-1}章.md"
```

**提取信息**：
- 剧情节拍列表
- 主视角角色
- 字数目标
- 场景位置
- 登场角色

### Step 2：按节拍生成

**生成规则**：

| 节拍 | 要求 | 字数占比 |
|------|------|---------|
| 1：XX 回到 XX | 适当场景描写，引入视角 | 15% |
| 2：众人在 XX 讨论 | 对话为主，推进剧情 | 25% |
| 3：XX 意识到 XX | 心理活动 + 外部行为 | 15% |
| 4：XX 开始调查 | 行动描写，展示方法 | 20% |
| 5：XX 发现 XX | 转折点，信息揭示 | 15% |
| 6：XX 质问 XX | 情绪渲染，冲突高潮 | 10% |

**视角切换标注**：
```
主视角切回到 XX
```

### Step 3：写作风格自检测

**自检清单**：

```markdown
## 写作风格自检

- [ ] 无比喻句（火山/心湖/石子等）
- [ ] 无情绪暗示（闪过/闪烁/带着）
- [ ] 无惰性状态描述（不容/不易/难以）
- [ ] 无动物比喻（像一只）
- [ ] 无平衡句式（不是…而是…）
- [ ] 无总结性名词化（这个动作/这句话）
- [ ] 无眼神描写
- [ ] 无嘴角描写
- [ ] 无指节描写
- [ ] 少用介词连词副词
```

**自检方法**：
- Grep 搜索禁止词语
- 人工审查禁止句式
- 统计介词连词副词频率

### Step 4：写作风格检测 Agent 审查

**调用**：
```bash
Task: writing-style-checker
参数：
- chapter: {NNNN}
- project_root: {项目根路径}
```

**审查结果处理**：
- 通过 → 进入 Step 5
- 不通过 → 返回修改后重新审查

### Step 5：输出章节

**输出格式**：
```markdown
# 第{NNNN}章 {标题}

---

{正文内容}

---

*本章字数：约{XXXX}字*
*下章预告：{预告内容}*
```

**验证**：
- 字数符合目标范围
- 剧情点全部覆盖
- 无细纲外内容
- 写作风格检测通过

---

## 输出示例

### 细纲示例

```markdown
# 第 0011 章 细纲

## 章节信息
- 章号：0011
- 标题：真相一角
- 字数目标：2300-2500
- 主视角：林越

## 剧情节拍

1. 林越回到住所（描写深夜森林的寂静）
2. 林越与幽夜讨论今天遇到的猎杀者
3. 林越意识到猎杀者公会已经盯上自己
4. 林越开始调查猎杀者公会的底细（通过询问幽夜）
5. 林越发现猎杀者公会在寻找纯血人族
6. 林越质问幽夜是否早就知道这件事（渲染被隐瞒的情绪）

---
**截止于此完结**
```

### 生成示例

```markdown
# 第十一章 真相一角

---

深夜。

森林静得出奇。

林越推开木屋的门，吱呀声在寂静中格外刺耳。他走到窗边，看着外面黑红色的月光。

"猎杀者公会的人今天追上来了。"

幽夜坐在桌边，水晶杖放在手边。那漩涡状的纹路缓缓转动。

"三个第二阶。"林越说，"他们知道我在这里。"

"他们一直在找你。"

"找纯血人族。"林越转过身，"或者说，找我身上的血。"

幽夜沉默了。

"你早就知道。"这不是问句。

"知道一部分。"幽夜的漩涡纹路转得更快了，"猎杀者公会……在收集纯血人族的血。"

"做什么？"

"不知道。"

林越盯着幽夜。许久，他开口：

"还有多少事是我不知道的？"

"很多。"幽夜说，"这个世界……比你想象的复杂。"

"比如？"

"比如天外之人。"幽夜站起身，走到窗前，"猎杀者公会只是其中之一。"

"还有谁？"

"还有很多。"幽夜望向远方，"有些……已经来了很久。"

林越没有说话。

他想起今天那三个人血红色的眼睛。想起他们说到"纯血人族"时贪婪的语气。

"我们需要更多源能结晶。"幽夜转过身，"你还需要升级。"

"升级之后呢？"

"活下去。"

很简单的一句话。

林越明白了。

---

*本章字数：约两千四百字*
*下章预告：林越决定主动出击！猎杀者公会的真正目的是什么？敬请期待第十二章！*
```

## 与 webnovel-write 的集成

### 调用关系

```
/webnovel-outline-write {chapter_num}
    ↓
Step 0: 预检（细纲/设定/前文）
    ↓
Step 1: 读取细纲与世界书
    ↓
Step 2: 按节拍生成正文
    ↓
Step 3: 写作风格自检测
    ↓
Step 4: writing-style-checker 审查
    ↓
Step 5: 输出章节
    ↓
Step 6: Data Agent 回写状态
```

### 模式差异

| 特性 | webnovel-write | webnovel-outline-write |
|------|----------------|------------------------|
| 输入 | Context Agent 任务书 | 细纲文件 |
| 创作自由度 | 中 | 低（严格遵循细纲） |
| 审查 | 4+Agent | writing-style-checker |
| 适用场景 | 常规章节 | 关键章节/伏笔章 |
