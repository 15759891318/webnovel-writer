# combat-scene-write (战斗场景写作 Skill)

> **Role**: 战斗场景写作专家，专门生成高质量、高画面感的战斗场景。

> **Philosophy**: 战斗描写要让读者感到"紧张 - 放松 - 更紧张"，在动作的间隙穿插情绪与环境的变化。

---

## 核心技巧库

### 技巧一：三回合原则（结构节奏）

**第 1 回合：试探与定调**
- 双方交手，展现风格（速度快/力道猛）
- 确立战斗紧张感
- 字数占比：30%

**第 2 回合：转折与环境改变**
- 加入环境破坏（打塌墙壁/下降到水下）
- 使战斗难度升级
- 字数占比：40%

**第 3 回合：终结与揭露**
- 使出大招，利用之前埋下的伏笔
- 反转局势，给读者"神转折"的爽感
- 字数占比：30%

**模板**：
```
【第 1 回合：试探】
[角色 A 出手] + [动作描写] + [对手反应]
[角色 B 反击] + [动作描写] + [环境效果]

【第 2 回合：转折】
[意外发生] + [形势逆转] + [心理描写]
[环境变化] + [难度升级] + [感官描写]

【第 3 回合：终结】
[伏笔揭示] + [大招使出] + [精准动词]
[效果描写] + [虚实结合] + [战斗结束]
```

---

### 技巧二：三要素比例（氛围/解说/打斗）

**氛围 30%**：
- 心理状态：心中一凛/后背发凉/怒火燃起
- 观众反应：亲友团/长辈/对手亲友团的看法
- 环境气氛：空气凝固/风雨欲来/万众瞩目

**解说 30%**：
- 招式背景：这招是……/此招源于……
- 厉害程度：这一击足以……/若是击中……
- 规则说明：比武规则是……/可以杀人么……

**打斗 40%**：
- 声音：噗嗤/轰隆/咔嚓
- 动作分解：侧身/挥刀/刺入
- 场面效果：血液喷涌/气浪翻滚/地面龟裂

**比例检查表**：
| 战斗类型 | 氛围 | 解说 | 打斗 |
|---------|-----|------|------|
| 遭遇战 | 25% | 25% | 50% |
| 势均力敌 | 30% | 30% | 40% |
| 越阶挑战 | 35% | 35% | 30% |
| 决战 | 40% | 40% | 20% |

---

### 技巧三：虚实结合（轻重鼓点）

**虚（轻鼓点）**：
- 效果描写：生命只剩五分之一
- 技能名字：战斗法师技能：龙牙
- 数据说明：消耗 5% 能量

**实（重鼓点）**：
- 画面描写：伞面伞骨竟然直接逆翻上去
- 声音描写：噗嗤/轰隆
- 细节描写：并拢的伞骨已经准准地刺到了……

**节奏模式**：
```
模式一：沙、沙、沙、沙、静、砰！
（连续虚写后接实写特写）

模式二：砰、沙、砰、沙、砰！
（虚实交替）

模式三：砰……砰……砰！
（越来越急促，最后收尾）
```

### 技巧四：视角转换（全景/中景/近景/特写）

**全景**（慢节奏，轻鼓点）：
- 大范围环境、气浪、观众
- 示例：夕阳西下，黑红色的余晖洒在两人身上，将影子拉得很长很长。

**中景**（中节奏）：
- 角色全身动作、移动轨迹
- 示例：林越也不甘示弱，掏出匕首，身形如电，冲向右边的扭曲地精。

**近景**（快节奏）：
- 半身动作、武器碰撞
- 示例：手腕一翻，刀刃的角度微调，噗嗤一声刺入地精的头颅。

**特写**（最快节奏，重鼓点）：
- 瞳孔、刀锋、伤口、指甲
- 示例：黑色的血液喷涌而出，溅在树叶上，发出滋滋的腐蚀声。

**转换模板**：
```
【全景】两人同时出手。
【近景】幽夜的水晶杖骤然亮起，一道梦幻流光划破空气。
【特写】"叽——！"地精惨叫一声。
【心理视角】林越心中一凛。
【特写】温热的鲜血溅到脸上，带着一股腥臭味。
【全景】森林深处，传来阵阵野兽的嘶吼声。
```

---

### 技巧五：五感并用（感官蒙太奇）

**视觉**：
- 颜色：血黑色/淡蓝色/银白色
- 光影：光芒大作/黯淡无光/闪烁不定
- 动作轨迹：划出一道寒芒/如同一道闪电

**听觉**：
- 兵器碰撞：当/噗嗤/咔嚓
- 惨叫：叽——！/啊——！
- 环境音：沙沙作响/呼啸而过

**触觉**：
- 疼痛：剧痛/刺痛/麻木
- 速度：快不及眼/骤然/瞬间
- 震颤：手掌发麻/身体震颤

**嗅觉**：
- 血腥味：刺鼻的腥臭味
- 焦糊味：空气中弥漫着焦糊的气息
- 腐臭：散发着腐臭

**味觉**：
- 血的味道：咸腥
- 汗水：苦涩

**五感组合示例**：
```
【视觉】黑色的血液喷涌而出。
【听觉】发出滋滋的腐蚀声。
【触觉】温热的鲜血溅到脸上。
【嗅觉】带着一股腥臭味。
【触觉】那种感觉，就像被强酸腐蚀，又像是有什么东西在骨头里爬行。
```

### 技巧六：精准动词库

**劈砍类**：
| 动词 | 力度 | 使用场景 |
|-----|------|---------|
| 劈 | 中 | 刀斧从上而下 |
| 斩 | 中上 | 利器横削 |
| 切 | 轻 | 快速划过 |
| 削 | 轻 | 擦过表面 |
| 剁 | 重 | 多次重击 |

**刺击类**：
| 动词 | 力度 | 使用场景 |
|-----|------|---------|
| 刺 | 中 | 尖锐物向前 |
| 戳 | 轻 | 短促向前 |
| 捅 | 重 | 深入内部 |
| 扎 | 中上 | 用力刺入 |
| 穿 | 重 | 贯穿而过 |

**打击类**：
| 动词 | 力度 | 使用场景 |
|-----|------|---------|
| 捶 | 中 | 拳头敲击 |
| 砸 | 重 | 重物下落 |
| 捣 | 中上 | 连续冲击 |
| 夯 | 重 | 大力砸击 |
| 擂 | 中 | 快速击打 |

**速度类**：
| 动词 | 速度 | 使用场景 |
|-----|------|---------|
| 窜 | 快 | 突然冲出 |
| 掠 | 极快 | 轻轻擦过 |
| 闪 | 极快 | 瞬间躲避 |
| 扑 | 快 | 猛扑过去 |
| 射 | 极快 | 如箭射出 |

**使用示例**：
```
❌ 原句：他打了他一拳，他又刺了他一剑。
✅ 修改：他以铁肘狠狠撞在对手锁骨上，那声音脆得像是在寒冬里折断的枯枝。
✅ 修改：匕首在空中划出一道寒芒，直刺地精的头颅。噗嗤。黑色的血液喷涌而出。
```

### 技巧七：暴力隐喻

**规则**：
- 将物理创伤与日常景象对比
- 增加描写强度和画面感
- 隐喻要贴切，不突兀

**隐喻库**：
| 创伤 | 隐喻 |
|-----|------|
| 骨头断了 | 像是在寒冬里折断的枯枝 |
| 血液喷涌 | 如同打开的蜂巢蜜 |
| 皮肤撕裂 | 像撕开一层薄纸 |
| 肌肉翻卷 | 如同绽放的花朵 |
| 神经颤抖 | 像被拨动的琴弦 |

**示例**：
```
❌ 原句：匕首捅进腹部。
✅ 修改：匕首捅进腹部时，她想起小时候捅开蜂巢蜜的颤动。

❌ 原句：骨头断了。
✅ 修改：那声音脆得像是在寒冬里折断的枯枝。

❌ 原句：血液喷出来。
✅ 修改：黑色的血液喷涌而出，溅在树叶上，发出滋滋的腐蚀声。
```

### 技巧八：环境利用

**环境定调**：
```
【战前】窗外，扭曲的树影在月光下摇曳，像无数只伸向天空的鬼爪。
【战中】两人顺势跌入破碎的窗户，在冷雨中继续翻滚。
【战后】夕阳西下，黑红色的余晖洒在两人身上，将影子拉得很长很长。
```

**环境互动**：
```
【利用环境】他侧身，刀锋贴着脊背擦过，在墙上划出一道刺眼的火星。
【环境破坏】一拳轰出，墙壁轰然倒塌，碎石飞溅。
【空间切换】从屋顶到室内，再到悬崖边。
```

### 技巧九：战斗速度把控

**快节奏**（小战斗）：
- 句子短促
- 少解说，多动作
- 省略氛围描写

**慢节奏**（重要战斗）：
- 句子延长
- 多解说，多氛围
- 详细描写心理

**速度调节示例**：
```
【快节奏】
掏出匕首。冲向右边的扭曲地精。
"老地方！"低喝一声。
噗嗤。黑色的血液喷涌而出。
地精倒地。

【慢节奏】
他缓缓抽出匕首，刀刃在月光下泛着森冷的寒光。
深吸一口气，他能感觉到心脏在胸腔里剧烈跳动，每一次搏动都像是在催促他前进。
"来吧。"声音低沉，却带着不容置疑的坚定。
他迈开步子，每一步都踩得很稳，脚下的落叶发出轻微的碎裂声。
```

### 技巧十：战斗心理与台词

**心理描写**：
```
【恐惧】心中一凛，后背渗出一层冷汗。
【愤怒】一股怒火从心底燃起。
【决心】他……不能丢下幽夜。那……就战吧。
【算计】心中快速盘算着对策。
```

**台词设计**：
```
【挑衅】"跑不了了。交出你的血，我们可以让你死得痛快一点。"
【宣示】"你的弱点……是头颅。"声音冰冷，如同来自九幽。
【嘴炮】"做你的梦。"说，声音冰冷。
【内心】"……"沉默了。他知道……幽夜说得对。
```

## 执行流程

### Step 1: 读取上下文

```bash
# 读取大纲
Read "大纲/第 1 卷 - 详细大纲.md"

# 读取前章
Read "正文/第{NNNN-1}章.md"

# 读取设定
Read "设定集/力量体系.md"
Read "设定集/怪物设定 - 后室风格.md"
```

### Step 2: 确定战斗类型

**战斗类型判断**：
| 类型 | 字数 | 节奏 | 三要素比例 |
|-----|------|------|-----------|
| 遭遇战 | 500-800 | 快 | 25%/25%/50% |
| 势均力敌 | 800-1200 | 中 | 30%/30%/40% |
| 越阶挑战 | 1200-1800 | 慢 | 35%/35%/30% |
| 决战 | 1800-2500 | 很慢 | 40%/40%/20% |

### Step 3: 设计战斗结构

**结构模板**：
```
【起因】为什么打？对手是谁？
【第 1 回合】试探与定调
【第 2 回合】转折与环境改变
【第 3 回合】终结与揭露
【战后】 victor 的反应/败者的下场/收获
```

### Step 4: 撰写战斗场景

**撰写顺序**：
1. 写氛围（心理/环境/观众）
2. 写解说（招式/规则/厉害程度）
3. 写打斗（动作/声音/效果）
4. 添加五感描写
5. 替换精准动词
6. 添加视角转换
7. 添加虚实结合
8. 利用环境互动

### Step 5: 自我检查

**检查清单**：
- [ ] 三回合结构是否完整？
- [ ] 氛围/解说/打斗比例是否合理？
- [ ] 是否有五感描写？
- [ ] 动词是否精准多元？
- [ ] 视角是否有转换？
- [ ] 虚实结合是否得当？
- [ ] 环境是否利用？
- [ ] 心理描写是否到位？

## 战斗描写生成示例

### 示例：遭遇战（快节奏）

**输入**：
```
章节：第 4 章
战斗类型：遭遇战
对手：3 只扭曲地精
 allies：幽夜
目标：展示配合战斗
```

**输出**：
```
【起因】
走了约莫十分钟。幽夜突然停下脚步，举起水晶杖。
"有怪物。"
"在哪？"
"前面。"压低身子，指了指前方。"三只扭曲地精。"

【第 1 回合：试探】
两人同时出手。
幽夜的水晶杖骤然亮起，一道梦幻流光划破空气，直射向左边的扭曲地精。
"叽——！"
地精惨叫一声，身体僵直，倒在地上抽搐了几下，不再动弹。
林越也不甘示弱，掏出匕首，身形如电，冲向右边的扭曲地精。
【查询中……】
【消耗能量：1%】
【弱点：头颅】
"老地方！"
低喝一声，匕首在空中划出一道寒芒，直刺地精的头颅。
噗嗤。
黑色的血液喷涌而出，溅在树叶上，发出滋滋的腐蚀声。

【第 2 回合：转折】
还剩一只。
那只地精看到同伴被杀，绿豆大的眼睛里满是恐惧，尖叫一声，转身就跑。
"想跑？"
冷笑一声，提刀追了上去。
但他没想到……
那只地精跑到一半，突然转身，咧开嘴，从嘴里吐出一团黑色的液体。
液体在空中散开，如同一张黑色的网，朝他扑面而来。
"小心！"
幽夜的声音从身后传来，带着几分焦急。
但已经晚了。
林越下意识抬手格挡，黑色液体溅到他的手臂上。
"嘶——"
倒吸一口凉气，额头瞬间渗出冷汗。
他的手臂……开始腐烂。
黑色的斑点以肉眼可见的速度蔓延，皮肉翻卷，露出下面森森的白骨。

【第 3 回合：终结】
【警告：宿主受到污染毒素攻击】
【当前污染值：10%】
"污染值？"
心中一凛，强忍着剧痛。
"系统，这玩意儿怎么清除？"
【无法清除】
【提示：污染不可逆】
沉默了。
手臂上的疼痛似乎都变得麻木。
污染不可逆。
那他……迟早会疯。

【战后】
"林越！"
幽夜快步跑过来，水晶杖发出的光芒照在受伤的手臂上。
"你怎么样？"
"没事。"
深吸一口气，强行压下疼痛。
"只是污染值增加了 10%。"
```

## 成功标准

- ✅ 三回合结构完整率 90% 以上
- ✅ 氛围/解说/打斗比例符合设定
- ✅ 100% 战斗有五感描写
- ✅ 动词精准，无"打/攻击"等普通词
- ✅ 视角转换每场战斗至少 3 次
- ✅ 虚实结合，轻重鼓点交替
- ✅ 环境利用至少 1 次
