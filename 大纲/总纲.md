# 总纲：《御兽：我能掌控诡异神明》

## 基本信息

| 项目 | 内容 |
|------|------|
| 书名 | 御兽：我能掌控诡异神明 |
| 题材 | 御兽流 + 克苏鲁 + 都市脑洞（国家后盾） |
| 预计字数 | 300 万字 |
| 预计章节 | 1000-1200 章 |
| 主角 | 林越（26 岁，陨石猎人） |
| 女主 | 无女主（专注事业） |

## 一句话简介

林越穿越到被克苏鲁污染覆盖的罔涅星，获得陨星核心，背靠蓝星国家，以御兽体系打破规则，最终解决污染根源拯救两界。

## 核心卖点

1. **御兽对象特殊**：不是传统宠兽，而是克苏鲁式的诡异生物
2. **国家后盾**：背靠国家机器而非个人奋斗，资源无限
3. **免疫污染**：唯一豁免污染的存在，可探索禁忌区域
4. **两界通道**：建立蓝星与罔涅星的永久通道，集体穿越

## 故事梗概

### 起点（1-100 章）

林越是蓝星的陨石猎人，在昆仑山收集陨石碎片时，被血黑色流星击中，穿越到罔涅星。

罔涅星是一颗被克苏鲁污染覆盖的超级星球，万族在疯狂与畸变中挣扎求生。林越获得陨星核心，能够与蓝星异界事务管理局建立联系。

初期，林越依靠国家支援在罔涅星生存，学习御兽体系，契约诡异生物，不断收集源能结晶提升实力。

### 发展（101-500 章）

林越的御兽体系大成，可独立作战。他发现罔涅星的污染根源与万年前的人族文明有关，而陨星核心正是人族留下的最后希望。

三大宗教（猩红教会、深海教团、万神教）视林越为异端，不断派出高手追杀。林越背靠国家，与宗教势力展开对抗。

林越发现万族对人族的敌意源于历史误会，开始尝试建立万族联盟，共同对抗污染。

### 高潮（501-800 章）

林越成就根源境，有能力触碰污染根源。他发现污染根源是外神沉睡的产物，而外神与万年前的人族灭亡有关。

三大宗教联手，企图唤醒外神，获得神力。林越率领国家舰队和万族联盟，与宗教势力展开最终决战。

### 结局（801-1000 章）

林越净化污染根源，外神被封印。罔涅星恢复生机，万族迎来新生。

林越建立蓝星与罔涅星的永久通道，两国人民开始交流。林越成为两界英雄，被万族传颂。

### 境界划分

| 境界 | 说明 | 代表战力 | 突破所需结晶 |
|------|------|---------|-------------|
| 污浊境（1 阶） | 刚接触污染，身体开始适应 | 普通万族 | 微光结晶×10 |

| 通神境（4 阶） | 能够借用外神力量 | 城邦主 | 古老结晶×10 |
| 化诡境（5 阶） | 身体部分诡异化 | 宗教主教 | 星之结晶×10 |

**核心规则**：
- 突破大境界必须使用**高一阶段**的结晶
- 低阶结晶无法替代高阶结晶突破
- 高阶结晶可以降级使用（1 块高阶=10 块低阶能量）

**结晶命名**（克苏鲁风格）：
- 微光→梦境→疯狂→古老→星之→外域→神性→盲目→混沌

## 创意约束

### 硬约束（不可违背）
1. 污染不可逆：被污染的物品/生物无法完全净化
2. 国家研发有风险：研发人员曾被污染导致性情大变
3. 陨石需要能量：所有功能都需要消耗源能结晶
4. 万族对人族的敌意：原始人族早已消失，主角是"异类"
5. 外神不可直视：直视会导致畸变

### 反套路规则
1. 主角不圣母：对敌人斩草除根
2. 国家不猜疑：立即相信并支援主角
3. 集体主义：最终依靠集体力量解决问题
4. 不靠个人英雄主义：关键时刻呼叫国家支援

### 主角缺陷
- 过于依赖国家：遇到危机第一时间联系国家
- 对无辜者心软：会出手相助

### 反派镜像
- 共享欲望：都想解决污染根源
- 相反道路：反派想利用污染成神，主角想净化污染

## 写作要求

### 禁止句式
- 无比喻句（火山/心湖/石子/湖面等）
- 无情绪暗示（闪过/闪烁/带着 xx 意味）
- 无惰性状态描述（不容/不易/难以）
- 无动物比喻（像一只 XX）
- 无平衡句式（不是…而是…）
- 无总结性名词化（这个动作/这句话）

### 写作规范
- 客观化、中立化叙述
- 展示而非讲述
- 尊重读者智力
- 少用介词连词副词
